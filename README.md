# lox-hermit-packages

Personal Hermit package repository for custom packages.

## Usage

To use packages from this repository, configure your project's `bin/hermit.hcl`:

```hcl
sources = [
  "https://github.com/cashapp/hermit-packages.git",
  "https://github.com/lox/lox-hermit-packages.git"
]
```

## Packages

- **pokerforbots** - Poker for Bots poker engine and bot framework
